# Server Requirements 
 -   PHP >= 7.2.0
 -   BCMath PHP Extension
 -   Ctype PHP Extension
 -   JSON PHP Extension
 -   Mbstring PHP Extension
 -   OpenSSL PHP Extension
 -   PDO PHP Extension
 -   Tokenizer PHP Extension
 -   XML PHP Extension
# Application features 
 - Membership system
 - Role system
 - Log system
 - Member link system
 - Public link sharing

# Installation 
Make the database connection from the .env file

    php artisan migrate    
    php artisan db:seed
